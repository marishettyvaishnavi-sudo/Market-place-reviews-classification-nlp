# Marketplace Reviews Classification using NLP

A Natural Language Processing project that automatically classifies marketplace customer reviews into sentiment categories using machine learning algorithms.

## 🎯 Project Overview

This project uses advanced NLP techniques to analyze and classify customer reviews from e-commerce marketplaces. The model automatically predicts whether a review is **Positive**, **Negative**, or **Neutral** based on the review text content.

### Problem Statement

Manually reviewing and classifying thousands of customer reviews is time-consuming and costly. This project automates the classification process using machine learning and NLP.

### Solution

Build an intelligent system that:
- Reads customer review text
- Processes and analyzes the text
- Automatically classifies sentiment
- Provides fast and accurate predictions

---

## ✨ Key Features

✅ **Text Preprocessing** - Cleaning, tokenization, lemmatization
✅ **Feature Extraction** - TF-IDF, Word2Vec embeddings
✅ **Multiple Algorithms** - Logistic Regression, Random Forest, SVM, Naive Bayes
✅ **Model Evaluation** - Accuracy, Precision, Recall, F1-Score
✅ **Visualization** - Confusion matrix, ROC curve, feature importance
✅ **Easy to Use** - Just load your review data and get predictions
